# 噩梦幸存者

san值归零：你永远迷失在了恐怖混乱的梦境中，再也无法醒来……
一局游戏时长：10分钟
成功通关：你大口喘着气醒来。原来又是一场噩梦。你将拖着疲惫的身体开始新的一天……

2.5D横板,主要左右移动，可小幅上下移动，躲避怪异人

人物立绘随san值降低有表情差分

背景逐渐黑化
- 繁华街
- 乡间小路晚上遛狗
- 学校走廊
- 地铁

怪物类型：
1. 会主动追赶玩家，需躲避。5秒后消失。被追上抠san值
2. 会主动追赶玩家，在接触时玩家站着不动就会消失。在接触时玩家正在移动则扣san值。
3. 站桩型，玩家不接近不会动。走远了就消失。靠近一定距离的话就会一直追着玩家（狂暴状态），直到触碰扣san。

1. 核心机制：地图块平铺（Tiling）与动态拼接游戏里的草地、地板，其实是由一张张固定大小的地图块（比如 $1024 \times 1024$ 像素的图片）拼接而成的。当你在游戏里一直往左走时，游戏引擎并不会真的去随机生成新的草地，而是把已经被你甩在身后的右边那块地图，“瞬移”到你的左边去接上。视觉效果：因为这些地图块的边缘是无缝衔接的（Seamless Texture），所以你在移动时，完全感觉不到地图被重复利用了，只会觉得地图大得没有边界。例外情况：有一些特殊地图（比如书库 Library），它在 $Y$ 轴（上下）是封死的，有墙壁阻挡，但 $X$ 轴（左右）依然是无限循环的。
2. 那地图上的特殊建筑/道具是随机的吗？既然基础地形是循环的，那地图上的植物、断壁残垣、甚至地上的特殊道具（如绿宝石、烤鸡、关卡特有武器）是怎么来的呢？固定生成的特殊建筑：在某些关卡，地图的循环中会穿插一些固定坐标的建筑。但这并不是随机的，而是作者提前在地图矩阵里写死的。你往一个方向走很久，会发现某些标志性建筑周期性地重复出现。随机刷新的小物件：像火把、光源、路灯这些可以打碎掉落物品的东西，它们是以玩家（镜头）为中心，在屏幕外随机坐标动态刷新（Spawn）出来的。当它们被玩家甩在身后很远时，就会被游戏系统销毁（Destroy），以此来节省内存。


1. 对特定生物与气味的生理厌恶（鱼、昆虫、腐败）
伊藤润二非常擅长利用“气味”和“异样动态”来引发读者的密集恐惧和反胃感。

对海洋生物与尸臭的恐惧 ——《鱼》（Gyo）
作品里带有诡异机械步行脚的鱼类涌上陆地。它探讨的是对未知海洋生物的巨物恐惧，以及死鱼身上散发的、无孔不入的瓦斯尸臭味。这种气味在漫画中被具象化为实质的气体，象征着人类对不可逆的污染与腐败的终极恐惧。

对昆虫及寄生异化的恐惧 ——《祖先们》（又名《头土》）
故事中历代祖先的头颅像毛毛虫一样连在一起，寄生在后代头上。这精准地抓住了人类对多足昆虫爬行、蠕动的生理厌恶，同时把“宗族血脉的压迫”实体化成了一条巨大的、由人头组成的肉虫。

2. 皮肤与触觉的极端不适（油腻感、密恐、身体异物）
他非常喜欢破坏皮肤的完整性，用来表现人类对“身体被弄脏、被侵蚀”的焦虑。

对极端油腻与排泄物的恶心 ——《三酸纤维素》（Glyceride，又名《油》）
这是伊藤润二最让人反胃的作品。整栋房子和空气里都飘着擦不掉的油烟，角色甚至靠喝油维生，最后脸上挤出火山喷发般的超级青春痘。它探讨的是对“不洁、黏腻、无法洗净”的触觉恐惧，将“油腻感”推向了精神污染的巅峰。

对皮肤千疮百孔的密集恐惧 ——《寒气》
人们的身体上突然长出密密麻麻、深不见底的小孔，寒风吹过肉体的小孔发出呜呜的声响。这直击人类对密集物体、肉体腐烂穿孔的本能抗拒。

3. 隐私侵犯与被凝视的焦虑（窥探、跟踪、丧失边界）
在现代社会中，空间狭小导致的“越界”常常让人发疯，伊藤润二对此有极深的刻画。

对隐私被无死角窥探的恐惧 ——《隔壁的窗户》
主角发现隔壁邻居古怪的老太婆，每晚都站在距离自己窗户极近的地方死死凝视着他，甚至把身体越过窗户伸过来。这是一种典型的“私人领地被侵犯”以及“被恶意凝视”的社交幽闭恐惧。

对现代偷窥与跟踪狂的焦虑 ——《窃窥狂》（又名《偷窥狂》）
房子和墙壁上被挖满了无数个小孔，无数只眼睛在暗处窥视着主角的一举一动。它放大了人们对“老大哥在看着你”或被变态跟踪、毫无隐私可言的现代都市恐慌。
