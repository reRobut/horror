Émilie Jaumain（朊病毒）、居里夫人（慢性辐射）以及 Karen Wetterhahn（二甲基汞渗透手套）
写游戏对话剧本，主人公是第一人称，她是一个刚进入顶尖研究所的年轻实习生，前程似锦，有一个非常支持她的伴侣。
第一章：日常工作，制作切片
第二章：有一天在收镊子的时候镊子不小心刺破了双层橡胶手套，出血两滴。研究所在研究一种未知病毒。主人公陷入恐慌。但带她的导师看了一眼，冷漠地说：“没事，这批样本灭过活，而且你戴了双层手套，消个毒继续干，明早开会要数据。”主角作为一个卑微的实习生，不敢反抗，不敢上报（怕被开除或延毕），只能强行安慰自己“应该没事”。主角觉得不对劲并陷入饱受折磨的怀疑。
第三章：工作崩坏。还在坚持工作。但是工作时逐渐开始产生恐怖的幻觉。比如显微镜下的细胞变成多个血红的眼睛，比如看到自己仿佛皮肤溃烂，比如开始出现轻微的耳鸣。在寝室洗澡时，主角会疯狂揉搓那个被刺伤的手指，直到搓出血来。在学校医务室检查，医生只是说：“你压力太大了，科研人员都这样，给你开点安神药。”
第四章：主角无法继续工作。休息在家，过了一段时间和平美好的日子。然后身上开始出现真的恶化症状。她开始出现非常严重的环境，视觉与语言系统崩坏，幻听，最后死亡。

“谨以此作献给所有在科学探索前沿，因安全疏漏或未知风险而默默牺牲的科研工作者们。”

场景三：乡间小路晚上遛狗（主题：失去挚爱）
现实痛苦：极度的孤独、宠物离世的创伤、对黑暗和未知的恐惧。
狗具有独立的 AI，它会朝着黑暗狂吠（预警怪物方向）。如果狗被怪物袭击，玩家的精神值（血条）会疯狂下降。

怪物设计：扭曲的枯树枝、以及长着人脸的流浪动物。
剧情/技能交互：随着游戏推进，狗绳会越来越长，代表主角“逐渐抓不住内心的寄托”。结局是主角终于走到路灯下，发现狗绳另一端其实什么都没有，只有一条项圈——原来现实中狗早就走了，这是他走不出的执念。

成功通关：你大口喘着气醒来，才想起爱狗已经去世了。
